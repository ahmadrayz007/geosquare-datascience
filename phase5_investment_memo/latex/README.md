## üêõ Troubleshooting

### Error: `! LaTeX Error: File 'tcolorbox.sty' not found.`
**Solution**: Install missing package
```bash
sudo apt-get install texlive-latex-extra
```

### Error: `! LaTeX Error: File 'fontawesome5.sty' not found.`
**Solution**: Install fonts package
```bash
sudo apt-get install texlive-fonts-extra
```

### Error: `! Unable to load picture or PDF file '../outputs/maps/grid_land_use.png'.`
**Solution**: Regenerate missing images (see "Regenerating Source Data" above)

### PDF is blank or has missing images
**Solution**: Check image paths are correct relative to latex/ folder
```bash
# From latex/ folder, verify images exist:
ls -lh ../outputs/maps/grid_land_use.png
ls -lh ../business_gap_analysis/outputs/business_gap_analysis.png
ls -lh ../../output/maps/rtrw_compliance_oku_only.png
```

### Compilation succeeds but PDF has wrong page count
**Issue**: Modifications to `.tex` file changed page breaks
**Solution**:
- Check `\vspace` adjustments
- Reduce text content if >2 pages
- Use 2-column layout (`\begin{multicols}{2}`)

## üìù Editing the Memo

To modify content, edit `oku_investment_memo.tex`:

```latex
\documentclass[12pt,a4paper]{article}  % DO NOT change font size below 12pt
\usepackage[margin=0.8cm]{geometry}    % Adjust margins if needed
\usepackage{multicol}
\setlength{\columnsep}{12pt}           % Column spacing

% ... content ...

\begin{multicols}{2}
  % Two-column layout starts here
  % LEFT COLUMN
  % RIGHT COLUMN
\end{multicols}
```

### Tips for maintaining 2-page limit:
- Use `\vspace{-4pt}` to reduce vertical spacing
- Use `\scriptsize` or `\small` for tables (but keep body text 12pt)
- Compress bullet lists with `\setlength\itemsep{0pt}`
- Use 2-column layout (`multicols`) for dense content
